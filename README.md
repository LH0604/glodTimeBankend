## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## 环境

```jsx
nodejs ---- 16.19.1
pnpm ------ 8.13.1
```
